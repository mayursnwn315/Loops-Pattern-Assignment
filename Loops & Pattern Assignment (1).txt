1.Write a program to print Alphabets A to H using pattern programming logic:
Ans-
java
Copy code
for (char ch = 'A'; ch <= 'H'; ch++) {
    System.out.print(ch + " ");
}


2.Write a program to print a triangle using star pattern programming logic:
Ans-
java
Copy code
int n = 5; // Number of rows
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= i; j++) {
        System.out.print("* ");
    }
    System.out.println();
}

Ans-
java
Copy code
